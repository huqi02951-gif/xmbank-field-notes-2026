## 维护规则

1. 新内容必须先增加来源 ID，再进入页面内容。
2. 私有来源仅写抽象代号和页码，不写本地绝对路径，不复制原文或图片。
3. 法规、产品、活动、外部平台和生活推荐都视为时效性内容；发布前复核，失效时删除而不是猜测更新。
4. APEX 重新同步时，只比较已批准的抽象结构；规则正文、客户字段和内部提醒默认拒绝迁移。
5. “允许公开”不等于已获最终批准；标记“条件允许”的条目必须完成表中人工复核。
