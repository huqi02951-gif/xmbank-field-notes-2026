# 实务研习院

面向 2026 台湾高校青年实习生的银行实务课程手册。它以观察、提问、整理与表达为能力主线，包含业务案例研讨、现场学习方法、课题研究方法、导师协作机制，以及一个脱敏、只读、不保存业务信息的 APEX 企业开户材料实验室。

## 本地运行

```bash
npm ci
npm run dev
```

完整发布检查：

```bash
npm run verify
```

## 内容与语言

`src/content/content.zh-CN.ts` 是正式正文入口；课程与现场方法拆分在同目录的聚焦模块。繁体显示由 OpenCC 在浏览器端按需转换，并保护银行专业术语。读者可以显示繁体，同时分别复制简体或繁体版本。

## APEX 边界

公开站仅保留脱敏材料分类、纯函数整理和内存交互。它不连接内部系统，不收集客户或企业信息，不上传、不保存、不导出，也不提供实时业务规则。教学结果必须由有权人员根据最新制度和具体场景复核。

## 发布

`main` 分支由 GitHub Actions 先执行内容审计、代码检查、类型检查、测试和构建，再将 `dist/` 发布到 GitHub Pages。生产构建只包含静态 HTML、CSS 和 JavaScript。
